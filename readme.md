## 🇬🇧 English

### 📌 Project Overview
This project presents an interactive **Power BI sales dashboard** built to analyze Adidas sales performance across regions, states, retailers, products, and sales channels.

The dashboard was designed to transform a raw dataset into a business-ready report that supports decision-making through KPI cards, geographic analysis, retailer comparisons, product performance tracking, and channel evaluation.

### 🎯 Objective
The main objective of this project was to create a clean and interactive dashboard capable of answering key business questions related to sales performance, profitability, product behavior, and channel contribution.

The project also focused on improving reporting structure, visual consistency, and storytelling quality so the final output could be included in a professional analytics portfolio.

### ⚙️ Process
The project was developed in **Power BI Desktop** using a structured workflow that included:

- Data import
- Data type validation
- Dashboard layout design
- Slicer configuration
- KPI creation
- Final insight extraction

### 📊 Dashboard Components
The dashboard includes the following elements:

- KPI cards for **Total Sales, Total Profit, Total Units, and Average Selling Price**
- Sales analysis by state through a **geographic map**
- Sales and profit comparison by **retailer**
- Sales distribution by **product category**
- Sales distribution by **channel**
- Interactive slicers for **region, retailer, sales method, and date**

### ❓ Questions Answered
This dashboard was built to answer the following business questions:

1. Which region generated the highest sales?
2. Which state showed the strongest sales performance?
3. Which retailer generated the most sales and profit?
4. Which product category performed best?
5. Which sales channel performed best?

### 💡 Key Insights
- **West** was the top-performing region, generating **269.94M in sales**.
- **California** was the leading state by a wide margin, with approximately triple the sales of most other states in the region.
- **Utah** remained significantly below the regional average, suggesting the need for deeper analysis and future improvement strategies.
- Retailer performance showed a strong alignment between **sales and profit rankings**, indicating a coherent and stable profitability structure across retailer partners.
- **Men's Street Footwear** was the best-performing product category, with **208.83M in sales** and **more than 80M in profit**.
- **Men's Athletic Footwear** outperformed the women's version by about **22%**, suggesting a potential opportunity for female-focused marketing strategies.
- **In-store** and **Outlet** channels showed similar sales participation, with **In-store slightly ahead**, while **Online** was the weakest channel.
- This suggests that digital promotion and stronger web/app visibility could help expand online sales nationwide.

This project demonstrates practical skills in:

- Visual design in Power BI
- KPI selection
- Business insight generation
- Data storytelling for professional portfolios

It also reflects the ability to transform raw data into a concise and professional narrative suitable for **GitHub, portfolio presentation, or interview discussion**.

## 🇪🇸 Español

### 📌 Descripción del proyecto
Este proyecto presenta un **dashboard interactivo en Power BI** diseñado para analizar el desempeño de ventas de Adidas a través de regiones, estados, retailers, productos y canales de venta.

El dashboard fue construido para transformar un dataset en bruto en un reporte visual orientado al negocio, capaz de apoyar la toma de decisiones mediante KPIs, análisis geográfico, comparación de retailers, evaluación de productos y revisión de canales de venta.

### 🎯 Objetivo
El objetivo principal del proyecto fue crear un dashboard limpio e interactivo que permitiera responder preguntas clave sobre ventas, rentabilidad, comportamiento de productos y aporte de cada canal.

También se buscó mejorar la estructura del reporte, la consistencia visual y la calidad narrativa para convertir el resultado final en una pieza sólida de portafolio profesional.

### ⚙️ Proceso
El proyecto fue desarrollado en **Power BI Desktop** siguiendo un flujo estructurado que incluyó:

- Importación del archivo
- Validación de tipos de datos
- Diseño del layout del dashboard
- Configuración de slicers
- Creación de KPIs
- Extracción de insights finales

### 📊 Componentes del dashboard
El dashboard incluye los siguientes elementos:

- Tarjetas KPI para **ventas totales, profit total, unidades totales y precio promedio de venta**
- Análisis de ventas por estado mediante **mapa geográfico**
- Comparación de ventas y profit por **retailer**
- Distribución de ventas por **categoría de producto**
- Distribución de ventas por **canal**
- Filtros interactivos para **región, retailer, método de venta y fecha**

### ❓ Preguntas respondidas
Este dashboard fue construido para responder las siguientes preguntas de negocio:

1. ¿Qué región generó el mayor volumen de ventas?
2. ¿Qué estado presentó el mejor desempeño en ventas?
3. ¿Qué retailer generó más ventas y profit?
4. ¿Qué categoría de producto tuvo el mejor desempeño?
5. ¿Qué canal de venta fue el más fuerte?

### 💡 Insights principales
- La región **West** fue la de mejor desempeño, con **269.94M en ventas**.
- **California** lideró con mucha diferencia frente a los demás estados de la región, alcanzando aproximadamente el triple de ventas que la mayoría.
- **Utah** quedó claramente por debajo del promedio regional, lo que sugiere la necesidad de profundizar en sus causas y pensar en mejoras futuras.
- El desempeño de los retailers mostró una fuerte alineación entre el ranking de **ventas y profit**, lo cual sugiere una estructura de rentabilidad coherente y estable entre socios comerciales.
- **Men's Street Footwear** fue la categoría de producto con mejor desempeño, alcanzando **208.83M en ventas** y **más de 80M en profit**.
- **Men's Athletic Footwear** superó a su variante femenina en aproximadamente **22%**, lo cual sugiere una oportunidad para impulsar estrategias de marketing orientadas al público femenino.
- Los canales **In-store** y **Outlet** presentaron una participación de ventas similar, con una ligera ventaja para **In-store**, mientras que **Online** fue el canal más débil.
- Esto indica una oportunidad clara para fortalecer la promoción digital y aumentar la visibilidad en sitios web y aplicaciones.
